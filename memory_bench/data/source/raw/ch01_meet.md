# ch01 相逢

```text
聪音：
啊，接通了！
摄像头····已经打开了吧····
能看得到吗？没问题吧？

我：我能看到你哦，

聪音：
是嘛，那就好.
你之前用过这个协作通话应用吗?
我稍微提前学了一下基本功能我还是可以教你的哦。

我：(点击互动)

聪音：你有什么想问的吗?

我：我想专注于创作。

聪音：那要不要试试番茄钟计时？听说劳逸结合更容易保持专注

我：我想写点笔记。

聪音：笔记功能也很好用哦，我也经常用它来记点小事。

我：我想整理一下要做的事情。

聪音:你可以试试用待办事项列表。我会把大学的课程、兴趣班的日程表都登记在里面

我：没问题

聪音：嗯，我知道了.有不少方便的功能用熟练了就好啦。我原以为会更让人紧张,没想到居然能和你这么自然地聊天,你呢？

我：也差不多吧。

聪音：太好了，你也这么想啊！总之，接下来也请你多多关照。那我们马上开始创作吧
```
