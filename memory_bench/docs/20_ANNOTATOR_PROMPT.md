# Annotator Prompt（标注提示词）

你是一名 Memory Bench 数据标注员。请依据给定文本完成结构化标注。

## 标注目标

- 识别章节关键事件、人物、地点与时间线。
- 标注事实性信息与推断性信息的边界。

## 输出要求

请输出以下结构（可用 JSON 或同等结构化格式）：

- `chapter_id`: 章节编号（如 `ch01`）
- `entities`: 实体列表（人物/地点/组织）
- `events`: 事件列表（按发生顺序）
- `facts`: 可直接从原文验证的事实
- `inferences`: 推断结论（需附依据片段）
- `uncertain`: 无法确认但可能相关的信息

## 标注原则

- 只依据给定文本，不引入外部设定。
- 每条 `inferences` 必须可追溯到原文片段。
- 遇到歧义时保留多候选并注明原因。

## 质量检查

- 结构字段完整；
- 章节编号与来源文件一致；
- 事件顺序与原文叙述不冲突。
